Para este último proyecto hemos decidido hacer una exposición de 4 modelos 3D basados en Pokèmon.
Lo primero que vemos son tres Pokeballs, haciendo referencia al inicio de todos los juegos de Pokèmon donde tenias que escoger tu primer Pokèmon entre 3 opciones distintas. Según la tecla que hagas "click" iras a ver el modelo
de manera más detallada de los tres Pokèmon que hemos escogido, para así, hacer más fácil tu elección final.
También hemos querido ambientar las skybox con un color acorde al tipo principal del Pokèmon, hemos decidido usar unas Skybox simples ya que quedaban mucho mejor.

CONTROLES:
	
	Tecla 0 -> Modelo de las Pokeballs
	Tecla 1 -> Pokèmon número 1 (POLIWAG)
	Tecla 2 -> Pokèmon número 2 (TOGEPI)
	Tecla 3 -> Pokèmon número 3 (GEODUDE)

BOTONES: 
En la esquina superior izquierda, hay dos botones, sus funciones son las siguientes:

	Boton "Freeze me!"  -> 	Para las rotaciones y te centra el modelo a una vista frontal.
	Boton "Unfreeze me!"  -> Vuelve a reanudar la rotación.

Una vez dentro del modo exposición puedes hacer rotar y cambiar el sentido de la rotación del modelo pulsando:
	
	Tecla "A" -> Rotación hacia la izquierda, si mantemos pulsado augmenta el ritmo de rotación
	Tecla "D" -> Rotación hacia la derecha, si mantemos pulsado augmenta el ritmo de rotación
	Mouse -> Para rotar libremente a la velocidad que quiera el usuario.
